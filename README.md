# DC-DC Buck Converter Project
A Simple Buck Converter Design done for Drexel's IEEE PCB Design Committee, I inttend to fabricate, assemble, and test these later.

## Power Specs
- Input voltage: 12V
- Output voltage: 3.3V 
- Current rating: 0-3A

## PCB Specs
- Board Dimension: 2 x 2 in^2
- Mounting Holes: 3mm
- Layers: 4
