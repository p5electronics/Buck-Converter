# DC-DC Buck Converter Project
A Simple Buck Converter Design done for Drexel IEEE

## Power Specs
  - Input voltage: 12V
  - Output voltage: 3.3V 
  - Current rating: 0-3A

## PCB Specs
- Board Dimension: 2 x 2 in^2
- Mounting Holes: 3mm
- Layers: 4

## Project Goal
- Fabricate, assemble, and test boards in spring quarter 2024
